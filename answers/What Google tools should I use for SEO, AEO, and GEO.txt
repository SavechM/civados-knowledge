# Core SEO & Visibility Tools

## Google Search Console

### What It Is

Google Search Console (GSC) is Google's direct communication channel with website owners.

It provides information about:

- Search performance
- Indexing
- Crawling
- Technical SEO
- Core Web Vitals
- Structured data

### Why It Matters

Google tells you:

- What keywords generate impressions
- What pages rank
- What pages are indexed
- What errors exist

### How To Use It

1. Verify website ownership
2. Submit sitemap
3. Monitor indexing
4. Review search queries
5. Track page performance
6. Monitor coverage issues

### Agency Use Cases

- SEO reporting
- Keyword discovery
- Index troubleshooting
- Crawl issue remediation

### Business Value

Without Search Console you are effectively blind to how Google sees your website.

---

## Google Analytics 4 (GA4)

### What It Is

Google's web analytics platform.

Tracks:

- Users
- Sessions
- Conversions
- Traffic sources
- User behavior

### Why It Matters

Answers:

- Where visitors come from
- What pages they visit
- Which pages convert
- What content performs best

### How To Use It

1. Install GA4 tag
2. Configure conversions
3. Connect Search Console
4. Build reports

### Agency Use Cases

- ROI tracking
- Lead generation reporting
- Funnel analysis
- Landing page optimization

### Business Value

Transforms website traffic into measurable business intelligence.
